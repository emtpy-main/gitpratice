# Git Practice Repo

This is a simple Node.js project for practicing Git operations.

## Files
- `index.js`: Main script with basic math functions.
- `utils.js`: Utility functions for greetings and math.

## How to run
```bash
node index.js
```

## Git Practice Ideas
- Add new functions to utils.js
- Modify existing code
- Create branches for features
- Merge branches
- Revert commits
